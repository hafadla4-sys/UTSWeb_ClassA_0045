# Mythic Boost - Website Joki Rank & Jual Beli Akun Mobile Legends

## Deskripsi Proyek

Mythic Boost adalah website yang menyediakan layanan Joki Rank Mobile Legends dan Marketplace Jual Beli Akun Mobile Legends. Website ini dibuat sebagai tugas Ujian Tengah Semester (UTS) mata kuliah Pemrosesan Web.

Website dirancang menggunakan HTML, CSS, dan JavaScript dengan tampilan modern, responsif, dan mudah digunakan oleh pengguna.

---

## Task UTS

Membuat sebuah website bertema layanan digital menggunakan HTML, CSS, dan JavaScript dengan ketentuan:

- Minimal 8 section
- Menggunakan file HTML terpisah
- Menggunakan file CSS eksternal
- Menggunakan file JavaScript eksternal
- Responsive pada perangkat desktop dan mobile
- Memiliki interaksi menggunakan JavaScript
- Memiliki desain yang menarik dan mudah digunakan


## Fitur Website

1. Navbar Navigasi
2. Hero Section
3. Tentang Kami
4. Layanan Joki Rank
5. Paket Harga
6. Marketplace Akun ML
7. Testimoni Pelanggan
8. FAQ (Frequently Asked Questions)
9. Form Kontak
10. Responsive Design
